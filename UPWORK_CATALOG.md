# Upwork Project Catalog — Paste-Ready Reference

Single reference for the 10 productized service listings to publish on Upwork Project Catalog.

**Why this matters:** The catalog is Upwork's "buy now" surface. Buyer picks a tier, clicks hire, no conversation needed. Each listing is a passive lead funnel. Highest-leverage revenue surface after the portfolio.

**Live preview / portfolio cross-link:** [samples.oncallautomation.ai](https://samples.oncallautomation.ai)

## How Upwork Catalog forms work

Each catalog listing has these fields (verify against Upwork's actual form when uploading):

| Field | Limit | Notes |
|---|---|---|
| **Project title** | ~100 chars | Must start with "You will get…" (Upwork requirement) |
| **Project description** | ~1000 chars | Main pitch — solution-first, client-side |
| **Tier names** (T1/T2/T3) | short | Use ones below or simplify |
| **Tier prices** | $ | Cancellable any month, no annual contracts |
| **Tier timelines** | days | Realistic — under-promise, over-deliver |
| **Tier inclusions** | bullets | What each tier ships |
| **Skills tags** | 5 max | Upwork's controlled vocabulary only |
| **FAQ** | optional | Add 3-5 if Upwork allows |
| **Sample work** | image upload | Use matching portfolio piece screenshot |

# Tomorrow's workflow

For each listing:
1. Open Upwork → Profile → Project Catalog → existing listing OR Create new
2. Paste **Title** exactly (must start with "You will get…")
3. Paste **Description** in the project description field
4. Set the **3 tiers** with prices + timelines
5. Paste **tier inclusions** as bullets in each tier's field
6. Add **Skills** (5 max from Upwork's controlled vocabulary)
7. Upload **Sample work** screenshot (matching portfolio piece)
8. Publish

**Expect 15-25 min per listing × 10 = 3-4 hours total.** Same time investment as today's portfolio. Bigger long-term revenue impact because catalog listings appear in Upwork's marketplace search — passive lead generation, not one-shot applications.
